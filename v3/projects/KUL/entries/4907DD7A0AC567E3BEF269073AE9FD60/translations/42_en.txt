【Right Page】
【Upper Section】
First, one paper box
This is called "papel" (paper) and "caixão" (box), and it contains the silver-like ornaments mentioned earlier.

Small buttons - fifteen
These are called "botão."
One wheel-like object made of shell
Soap - a little
This is like washing powder.
One brush
This is called "brush."

【Lower Section】
First, two cord-like items
These are called "tirante" and are suspenders for trousers, worn over the shoulders.

First, one cotton bag
This is sewn by Zensuke from fabric scraps.

First, one leather small box
Items contained inside:
Leather strop
This is for sharpening razors, but I don't know the name.
One razor
This is called "navalha."

【Left Page】
【Upper Section】
One needle
This is called "agulha."
Slate pencil
This is a writing stone for black slate used instead of paper, but I don't know the name.

First, some thread
This is like hemp thread and is called "fio."

First, one horn
This is called "chifre," a cow's horn that Zensuke crafted into a fishing tool.

First, two rings
I don't remember what these are called.

【Lower Section】
One silver spatula
This is called "limpa" and is used for cleaning the tongue.

One pen holder
This is for making pens to write horizontal script.

One file
This is called "lima" and is used for filing nails.

One comb
This is called "pente." The original comb contained within was worn out, so the current comb was received as a replacement in Zhapu, China.

First, some matchsticks
These are called "fósforo." When rubbed against the bottom of a tube containing chemicals at the tip, they produce fire.

Fifth, Kokushi (Black Teeth)
In the secret teachings, this is Fudō (Acala); in the Dainichi Shōkyō, this is named "Giving Black"

Sixth, Tahatsu (Much Hair)  
In the secret teachings, this is Fudō (Acala); in the Dainichi Shōkyō, this is named "Giving Black"

Seventh, Muensoku (Never Satisfied)
In the secret teachings, this is Munōshō (Invincible); in the canonical scriptures, this is named "Unattached"

Eighth, Jiyōraku (Bearing Necklaces)
In the secret teachings, this is Kannon (Avalokiteśvara); in the canonical scriptures, this is named "Bearing Flowers"

Ninth, Sōtai
In the secret teachings, this is Monju (Mañjuśrī); in the canonical scriptures, this is named "What Place"

Tenth, Datsuissai Shujō Seiki (Stealing the Life Force of All Sentient Beings)
In the secret teachings, this is Datsuissai Kūkichini
In the Lotus Sutra, this is the deity Uga
Also Daijizai Bosatsu (Maheśvara Bodhisattva)
In the canonical scriptures, this is named "All Essence"
